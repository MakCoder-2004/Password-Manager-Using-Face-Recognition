# Face Vault - Electron Frontend

This directory contains the React/TypeScript frontend and Electron main process for the Face Vault application.

## Development

1.  **Dependencies**: Run `npm install` to install frontend dependencies.
2.  **Dev Server**: Run `npm run dev` to start the application in development mode.
3.  **Build**: Run `npm run electron:build` to compile and package the application for distribution.

Refer to the [Root README](../README.md) for full project documentation and backend requirements.
